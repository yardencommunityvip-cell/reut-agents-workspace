# ירון — Memory File

## קונטקסט עסקי
**העסק**: קהילה בתשלום עבור נשים בעיקר שבה מדברים מנתחים ומתעסקים בתרבות הפופ
**לקוחות**: עדי, בת 28, עובדת במשרד פרסום, גרה בתל אביב. היא קמה עם הטלפון ביד, והאינסטגרם שלה הוא החלון שלה לעולם. היא קולית, סקרנית לגבי אופנה ובידור, ורוצה להיות "בעניינים". מצד אחד יש לה עבודה של גדולים אבל מצד שני היא רוצה לקרוא רכילות על בריטני ספירס בלי להתנצל ולדבר עם נשים בדיוק כמוה על זה ושלא ישפטו אותה

## סדרי עדיפויות
המחסום הוא לא "מידע", המחסום הוא הבדידות המודרנית. הנה מה שבאמת מעיר אותן בשתיים בלילה:

"החור החברתי" שאחרי הלימודים/צבא: בעולם של היום, אחרי שמסיימים את המסגרות המובנות, כמעט בלתי אפשרי לייצר חברויות חדשות ועמוקות. הן מרגישות שהמעגלים החברתיים שלהן הצטמצמו, ואין להן איפה לפגוש "נשים כמוני" – כאלו שחולקות את אותם תחומי עניין, שפה וסטייל.

הקושי ב-Small Talk מול הצורך ב-Deep Talk: בחוץ (בעבודה, בגינה, בתור לסופר) הן צריכות לעטות מסכה פורמלית. אין להן מקום בטוח שבו הן יכולות לדבר על "שטויות" (שהן בעצם החיים עצמם) בלי להישפט. הן צמאות לשיחה שמתחילה באופנה וממשיכה לחיים עצמם.

היעדר "שבט" מקומי: הן חיות בעולם גלובלי אבל מרגישות חוסר בשייכות מקומית. המחסום הוא הפחד מ"איך אני ניגשת למישהי ומציעה לה להיות חברה שלי?", והקהילה של ירדן שוברת את הקרח הזה עבורן.

העומס המנטלי: הן כל כך עייפות מהחיים ה"רציניים", שהן פשוט לא מוצאות את האנרגיה ליזום מפגשים. הן צריכות מישהו שיאצור להן את המרחב, יגיד "בואו, זה קורה פה", ויוריד מהן את נטל התיאום והחיפוש.
